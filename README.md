# Obs Stream Widgets

## Project setup
```
yarn
```

### Run APP and server
```
yarn start
```

```
WIP
```

```js
  widgets: {

    'chat': require('@obs-stream-widgets/widget-chat'),
    'your-widget': require('@obs-stream-widgets/your-widget'),
    ...
  }
}

```
